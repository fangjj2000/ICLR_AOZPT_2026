# AOZPT_ICLR2026
## For text_to_text task
### Create conda environment:
> conda env create -f ICLR2026/text_to_text/environment.yml -n t2t
> 
> conda activate t2t
>
### Run text_to_text task:
> bash ICLR2026/text_to_text/gpt4/zo/run.sh

## For text_to_image task
### Create conda environment:
> conda env create -f ICLR2026/text_to_image/environment.yml -n t2i
> 
> conda activate t2i
>
### Run text_to_image task:
> bash ICLR2026/text_to_image/run.sh
